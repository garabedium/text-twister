    Dictionary by Wordnik | GitHub
    • A tribute to the anagram game Text Twist.
    • Built with pure Javascript
    • Leverages Wordnik Dictionary API

    How to Play

    • timer Solve as many words as you can before time runs out.
    • error Word must be a minimum of three letters.
    • trending_up Solve a six letter word and move on to the next level.

    Game Controls

    • space_bar Spacebar: scramble letters
    • keyboard_return Enter / Return: solve word
